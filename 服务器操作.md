## 1、更新game-control操作

1. 登录161.189.164.190nginx服务器

2. 定期清理log/nginx 下日子文件

   ```
   #写空文件
   > filename
   ```

3. 进入docker

4. 查看nginx配置

5. 基础配置再nginx.conf

6. 修改配置（每次主注掉一台机器并重启）

7. nginx -t 测试配置文件

8. 重载配置nginx -s reload

   

## 2、上传images到镜像仓库

1. AWS里查ERL
2. 找到对应镜像名称
3. 有相关推送教程
4. 登录aws，过滤saml
5. 通过脚本将saml中的key解析出来
6. 用key登录镜像仓库
